Sistema Académico en Python (SICEU)

Este proyecto implementa una aplicación académica inspirada en el sistema SICEU (UNEFA), desarrollada en **Python** utilizando las librerías **tkinter**, **customtkinter** y **time** para la interfaz gráfica y la gestión de horarios.  
El sistema gestiona roles de **Profesor**, **Coordinador** y **Estudiante**, ofreciendo funcionalidades específicas para cada uno.

Funcionalidades principales

Rol Profesor
- Visualizar únicamente las materias asignadas.
- Consultar la lista de estudiantes inscritos en sus materias.

 Rol Estudiante
- Registro de estudiantes en el sistema.
- Inscripción de materias disponibles.
- Visualización del horario académico.
- Generación de constancia de inscripción.
- Consulta del récord académico (notas).

 Rol Coordinador
- Registro y administración de estudiantes.
- Asignación de materias a los profesores.
- Gestión de la oferta académica.

 Tecnologías utilizadas
- **Python 3**  
- **tkinter** → interfaz gráfica básica.  
- **customtkinter** → interfaz gráfica moderna .  
- **time** → gestión de horarios y control de tiempo.
